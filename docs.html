🎵 Musiqalar
🔒 Barcha musiqa endpointlari uchun Authorization: Bearer TOKEN header kerak!
POST /api/music Musiqa yuklash 🔒 ▼
⚠️ Content-Type: multipart/form-data (JSON emas!)
Form Data
ParametrTuriTavsif
audio shart file Audio fayl (mp3, wav, ogg, flac, aac, m4a) — max 50MB
title shart string Musiqa nomi
artist shart string Ijrochi nomi
genre ixtiyoriy string Janr (default: "Boshqa")
duration ixtiyoriy number Davomiyligi (soniyalarda)
cURL misol
curl -X POST BASE_URL/api/music \
  -H "Authorization: Bearer TOKEN" \
  -F "audio=@song.mp3" \
  -F "title=Mening qo'shig'im" \
  -F "artist=Muhammad" \
  -F "genre=Pop"
Javob
{
  "success": true,
  "message": "Musiqa muvaffaqiyatli yuklandi",
  "data": {
    "_id": "...",
    "title": "Mening qo'shig'im",
    "artist": "Muhammad",
    "genre": "Pop",
    "plays": 0
  }
}
GET /api/music Barcha musiqalar 🔒 ▼
Query Parametrlar
ParametrTuriTavsif
search ixtiyoriy string Nomi yoki ijrochisi bo'yicha qidirish
genre ixtiyoriy string Janr bo'yicha filtr
page ixtiyoriy number Sahifa raqami (default: 1)
limit ixtiyoriy number Har sahifada nechtasi (default: 20)
Misol
GET /api/music?search=love&genre=Pop&page=1&limit=10

Javob

}
GET /api/music/:id/stream Musiqani tinglash (stream) 🔒 ▼

Audio faylni stream qiladi. Brauzerda to'g'ridan-to'g'ri tinglash mumkin. Har safar tinglanganda plays soni 1 ga oshadi.

HTML misol
<audio controls src="BASE_URL/api/music/ID/stream"></audio>
💡 Qo'llab-quvvatlanadi: Range headers (seeking uchun), audio/mpeg content type
❌ Xatolik formati
{
  "success": false,
  "message": "Xatolik tavsifi"
}
